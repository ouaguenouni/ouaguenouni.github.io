This is a speaker note — only you see it in the speaker view (press S). It gives you notes, a timer, and a preview of the next slide.

Why not Beamer?

  • Slides are Markdown, not LaTeX boilerplate
  • Save → refresh, no pdflatex round-trip
  • Real LaTeX math still works: $e^{i\pi} + 1 = 0$
  • Code highlighting, images, tables, web embeds
  • One keystroke to export a PDF handout

The math uses the exact same dollar-sign delimiters as your articles, and is compiled with the same pipeline — so anything you wrote in KaTeX pastes straight in.
